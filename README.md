# SNAKESNAKE

-Sobre el Juego
Esta es una versión adaptada del clasico Snake, diseñada para mantener la esencia pura del juego original que cautivó a generaciones.

-Dinámica 
  -Mecánica Original: La jugabilidad se mantiene intacta; la serpiente se desplaza por una cuadrícula, crece al alimentarse y el juego termina si colisionas con las paredes o con tu propio cuerpo.
  -Curva de Dificultad: Mantiene el ritmo constante y desafiante del juego tradicional.
  -Controles Intuitivos: Movimiento fluido mediante las flechas de dirección (Arriba, Abajo, Izquierda, Derecha).

-Interfaz 
Uso de tipografía estilo Pixel Art ('Press Start 2P') para evocar la nostalgia de las consolas de 8 bits.
Incorporación de efectos de sonido inmersivos cada vez que la serpiente consume alimento.

-Tecnologia Utilizada
  -Nodejs (V24.13.1)
  -React (V19.2.4)
